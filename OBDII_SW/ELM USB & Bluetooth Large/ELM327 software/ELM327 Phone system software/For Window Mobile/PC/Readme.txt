Usage
--------------------------------------------------------------

To view OBD sensor data on your Windows PC, run:

  OBDGraph.exe

Choose the "Open OBD Data..." menu item, then select the file
"OBD Data Sample.pdb" to see sample data.

You can use OBD Graph to display your own data using the
following procedure:

OBD Gauge for Palm:

  Select the "Start Recording" menu item in OBD Gauge to
  begin capturing data. After performing a HotSync, select
  the "Open OBD Data from Palm" menu item in OBD Graph.

OBD Gauge for Pocket PC:

  Select the "Recording", "Start" menu item in OBD Gauge to
  begin capturing data. In Microsoft ActiveSync on the PC,
  select the "Tools", "Options" menu item. Check the "Files"
  option in the "Sync Options" tab page. After synchronizing
  the Pocket PC, select the "Open OBD Data from Pocket PC"
  menu item in OBD Graph.

If an error dialog appears when attempting to start OBD Graph,
you may need to install Microsoft .NET Framework 1.1.
The .NET Framework is available through "Windows Update":

  http://windowsupdate.microsoft.com/

Or as a separate download:

  http://msdn.microsoft.com/netframework/downloads/updates/
