Installation
--------------------------------------------------------------

To install OBD Gauge on the Pocket PC, run "setup.exe".
Microsoft ActiveSync is required.

The Microsoft .NET Compact Framework 1.0 must be installed on
the Pocket PC. It can be downloaded from Microsoft:

http://msdn.microsoft.com/netframework/downloads/updates/
